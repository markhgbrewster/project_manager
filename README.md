# README

This is a simple rails app that implements a basic project tool. 
It tracks changes to a projects status and allow users to comment on the project 
and tracks comments

## Uses

* Ruby version: 3.2.1
* Rails version: 7.0.4
* PostgreSQL: 14.7

## To Run

* bundle
* rake db:create
* rake db:migrate
* ./bin/dev
* navigate to http://localhost:3000


## To Run the specs
* rspec spec
